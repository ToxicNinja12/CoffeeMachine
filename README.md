# Coffee Machine

This is a learning project intended to create a CLI tool that simulates a coffee vending machine.

### 🚀 Features (todo 🙂):

- Tracks remaining ingredient supplies.
- Accepts simulated coin inputs.
- Ensures ingredient availability for each order.
- Generates report on available balance and ingredients.

### 📦 How to run

1. Clone this repo:
    ```shell
    git clone https://github.com/ToxicNinja12/CoffeeMachine.git
    cd natours
    ```
2. Install dependencies with:
    ```shell
    pip install -e .
    ```
3. Run:
   ```shell
   coffee
   ```
